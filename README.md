# StatusCode

Enums and utilities around status codes.

## Requirements

- `node >= 7.10`
- `typescript >= 2.4`

## Installation

`npm i @bluejay/status-code [--save]`

## Usage

```typescript
import * as StatusCode from '@bluejay/status-code';

StatusCode.OK; // 200
```

Alternatively, you can use the named export.

```typescript
import { StatusCode } from '@bluejay/status-code';

StatusCode.OK; // 200
```
